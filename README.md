# 运单全流程管理系统 V3

> 录单 → 扫描品控 → 异常上报 → 分级审批 → 执行联动 —— 运单全生命周期管理

## 系统简介

V3 是运单全生命周期管理平台，独立部署的系统，通过接口服务与 V2 数据互通。覆盖运单从入仓到交付完成的完整链路。

## 快速开始

```bash
# 安装依赖
npm install

# 初始化数据库
npx prisma db push

# 初始化种子数据（创建用户、规则、200+ 工单）
npm run db:seed

# 启动开发服务器
npm run dev

# 访问 http://localhost:3000
```

## 技术栈

- **框架**: Next.js 16 App Router + TypeScript
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: Prisma
- **认证**: NextAuth.js v5
- **UI**: Tailwind CSS
- **部署**: Vercel

## 核心功能模块

### 模块零 · 扫描品控
- SKU 扫描录入 → 品控规则引擎 → 通过/异常判定
- 品控暂扣（批次锁定）→ 自动创建工单
- 扫描幂等性（重复扫描追加记录不创建工单）
- 品控主管"误判快速放行"

### 模块一 · 异常工单上报
- 手工上报物流/品控异常
- V2 接口实时校验运单真实性
- 归属校验 + 重复上报检测
- AI 辅助异常分类（可选）

### 模块二 · 分级审批引擎
- 完整两级审批状态机
- 并发冲突保护（操作令牌机制）
- 审批人离职/禁用兜底
- 超时自动流转
- 自批自核防护

### 模块三 · 执行联动
- 审批通过后联动赔付/库存/退仓
- 数据库事务保证一致性
- 赔付记录可追溯至审批记录
- 品控赔付 = 向供应商追偿 | 物流赔付 = 赔付客户

### 模块四 · 工单列表与追踪
- 按状态/类型/运单号筛选
- 分页 + 数据来源标注
- 审批历史审计日志

### 模块五 · 接口监控
- V2 服务健康状态
- 接口调用日志
- 降级方案
